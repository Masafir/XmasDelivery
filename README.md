## Before start 
### Create a new branch
` git checkout -b LeNomDeMaBranche `
## When the work is finish
###  Add changed files
` git add .`
### Commit changed
` git commit -m 'le nom du commit' `
### PULL OTHER CHANGE 
` git pull origin main `
### Resolve conflicts if they are one
### Push change
` git push --set-upstream origin LeNomDeMaBranche`
